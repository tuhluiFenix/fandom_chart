# fandom_chart

Этот проект использует API Ao3 для составления диаграммы и сравнения трех фандомов
## Описание

Скрипт выполняет следующие действия:
1. Делает запрос к Ао3 API для поиска работ, которые отсортированы по фандомам, кроссоверы отброшены
2. Обрабатывает полученные данные
3. Создаёт интерактивную круглую диаграмму с использованием Plotly Express
4. Отображает график

## Требования

Для работы проекта необходимо установить:
- Python 3.6+
- Библиотеку `AO3` (`pip install AO3`)
- Библиотеку `plotly` (`pip install "plotly[express]"`)
